🔥 Jak nastavit Firebase sync — krok za krokem

Celé nastavení zabere cca 15 minut. Vše je zdarma.

KROK 1 — Vytvoř Firebase projekt
1. Jdi na console.firebase.google.com
2. Klikni „Add project"
3. Název: svadba-2027
4. Google Analytics: vypni → Create project
KROK 2 — Vytvoř Realtime Database
1. Levé menu: Build → Realtime Database
2. Create database
3. Lokace: europe-west1 (Belgium)
4. Security rules: Start in test mode → Enable
KROK 3 — Získej konfiguraci
1. Project Overview → ikona </> (Web)
2. App nickname: svadba-web → Register
3. Zkopíruj celý blok firebaseConfig = { ... }
KROK 4 — Vlož config do souboru na GitHubu
1. GitHub → klikni na index.html
2. Ikona tužky ✏️ → Edit
3. Najdi apiKey: "TVUJ_API_KEY"
4. Nahraď celý blok FIREBASE_CONFIG svými hodnotami
5. Commit changes → Vercel nasadí za 30 sekund
✅ Jak upravovat appku v budoucnu
GitHub → klikni na soubor → ✏️ Edit → uprav → Commit
